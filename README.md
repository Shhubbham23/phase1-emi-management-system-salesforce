# EMI Management System (Salesforce)  

A Salesforce mini-project to manage and calculate EMI schedules for loans using **Lightning Web Components (LWC)** and **Apex Classes**.  

---

## Features  

- **EMI Schedule Table**: Dynamic display of EMI installments  
- **Generate EMI Button**: Quickly calculate and populate EMI schedules  
- **Loan EMI Container & Summary Card**: View loan details and summaries  
- Backend logic handled by `EMIScheduleController` & `GenerateEmIController` Apex classes  
- Modular, reusable, and portfolio-ready LWC components
